# XAMPP Setup Instructions for MediaRecs

## Step 1: XAMPP Installation
1. Download and install XAMPP from https://www.apachefriends.org/
2. Start Apache and MySQL services from XAMPP Control Panel

## Step 2: Project Setup
1. Copy all project files to: C:\xampp\htdocs\mediarecs\
2. Files are already configured for XAMPP/MySQL

## Step 3: Database Setup
The database will be created automatically when you first visit the site.

## Step 4: Access the Site
Open browser and go to: http://localhost/mediarecs/

Note: API keys are already configured and working.
